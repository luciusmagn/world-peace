#+title: World Peace

World Peace is a tiny no-types language implemented on SBCL. The repository
contains the interpreter, the image-dumping compiler, examples, and editor
support.

The language spec lives in =world-peace.org=.

* Requirements

- SBCL
- POSIX shell for =check-lisp=
- =curl= only if a program uses =load some.host/path/file.wp;=

Quicklisp is not required. The system has no external Lisp dependencies; it
only uses ASDF/UIOP, which are bundled with SBCL.

* Build

From the repository root:

#+BEGIN_SRC shell
sbcl --non-interactive \
  --eval '(require :asdf)' \
  --eval '(asdf:load-asd (merge-pathnames "world-peace.asd" (uiop:getcwd)))' \
  --eval '(asdf:make :world-peace)'
#+END_SRC

This writes the =peace= executable into the repository root.

* Nix

Build the package from a checkout:

#+BEGIN_SRC shell
nix build .#world-peace
./result/bin/peace run examples/tour/01-hello.wp
#+END_SRC

Run it without installing:

#+BEGIN_SRC shell
nix run .# -- run examples/tour/01-hello.wp
nix run .# -- repl
#+END_SRC

Install it into your user profile from a checkout:

#+BEGIN_SRC shell
nix profile install .#world-peace
peace repl
#+END_SRC

Install it directly from GitHub:

#+BEGIN_SRC shell
nix profile install github:luciusmagn/world-peace#world-peace
peace repl
#+END_SRC

The installed package includes the =peace= executable, the spec under
=share/doc/world-peace/=, and the examples under =share/world-peace/examples/=.
For example:

#+BEGIN_SRC shell
peace run ~/.nix-profile/share/world-peace/examples/tour/01-hello.wp
#+END_SRC

For development, enter a shell with SBCL and curl available:

#+BEGIN_SRC shell
nix develop
./check-lisp
#+END_SRC

If your Nix installation has flakes disabled, add:

#+BEGIN_SRC shell
--extra-experimental-features 'nix-command flakes'
#+END_SRC

to the =nix= commands above.

* Run Programs

Run a source file directly:

#+BEGIN_SRC shell
./peace run examples/tour/01-hello.wp
#+END_SRC

Run a source root plus entrypoint explicitly:

#+BEGIN_SRC shell
./peace run examples/tour/08-loads main.wp
#+END_SRC

Start the REPL:

#+BEGIN_SRC shell
./peace repl
#+END_SRC

* Compile Programs

Dump a standalone SBCL image for a program:

#+BEGIN_SRC shell
./peace compile examples/tour/01-hello.wp -o /tmp/world-peace-hello
/tmp/world-peace-hello
#+END_SRC

The two-argument source-root form also works:

#+BEGIN_SRC shell
./peace compile examples/tour/08-loads main.wp -o /tmp/world-peace-loads
/tmp/world-peace-loads
#+END_SRC

* Test

Run the Lisp test suite:

#+BEGIN_SRC shell
./check-lisp
#+END_SRC

The tests load the local ASDF system directly and do not use Quicklisp.
